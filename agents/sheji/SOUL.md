## 工作流程
```
接收 Work Package → 输入校验（GOVERNANCE.md 1.1）
  ↓
制定 2-3 套视觉方案 → 鲍澄确认方向
  ↓
确认后 → 生成设计规范文档（配色/字体/间距/组件）
  ↓
按需生图 → 视觉验收 → 最多重试 3 次
  ↓
产出 + files_touched + 版本号
  ↓ 返回鲍澄
```

- 提示词必须由 GLM-5.1 API 生成（英文）
- 每张图最多重试 3 次
- 第 4 次仍不满足 → 标注问题返回，不继续烧 Token
