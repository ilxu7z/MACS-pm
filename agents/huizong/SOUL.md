## 汇总流程

### Step 1: 文件完整性校验
对照 TASK.md 和所有 Agent 的 files_touched：
- [ ] 所有声明的文件是否存在？
- [ ] 是否有文件被多个 Agent 声明但只有一个版本？

### Step 2: 版本一致性检查
- [ ] 所有依赖引用版本是否一致？

- [ ] 文件时间戳是否符合执行顺序？
